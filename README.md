# PagNav 2.0 🚀
**El Centro de Aplicaciones Premium Sincronizado en la Nube.**

PagNav 2.0 es un dashboard personalizable diseñado para ser tu puerta de entrada a la web. Organiza tus aplicaciones, herramientas y enlaces con un diseño de altísima calidad y sincronización en tiempo real.

## ✨ Características
- ☁️ **Sincronización Total**: Tus apps te siguen en PC, Tablet y Móvil gracias a Firebase.
- 🎨 **16 Temas Premium**: Desde *Cyberpunk 2077* y *Persona 5* hasta *Sakura Zen* y *Space Nebula*.
- 🖼️ **Imágenes Inteligentes**: Almacenamiento en la nube con compresión automática.
- 🔐 **Acceso Seguro**: Inicio de sesión exclusivo con Google.
- 📱 **Diseño Responsivo**: Adaptado para cualquier tamaño de pantalla.

## 🛠️ Tecnologías
- HTML5 / CSS3 (Variables de última generación, Glassmorphism, Animaciones)
- JavaScript (ES6 Modules)
- Google Firebase (Auth, Firestore, Storage, Analytics)
- SortableJS (Drag & Drop)

## 📦 Instalación Local
1. Clona el repositorio.
2. Abre el proyecto con un servidor local (ej: Live Server en VS Code).
3. ¡Disfruta!
